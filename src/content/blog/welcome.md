## Writing a real post

Create a new Markdown file under `src/content/blog/` with front matter:

```yaml
---
title: "Your title"
description: "One-line summary for listings and SEO."
pubDate: 2026-07-10
draft: false
---
```

Set `draft: false` to publish. Standard Markdown is supported: headings, lists,
links, and fenced code blocks.
